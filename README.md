### 🪔 Diwali Sales Analysis – Python Project

##### Project Overview:
This project analyzes Diwali sales data using Python to uncover customer purchasing behavior, sales trends, and business insights. The objective is to perform exploratory data analysis (EDA), clean the dataset, and generate actionable insights to support marketing and sales strategies.

##### Project Objectives:
1. Perform data cleaning and preprocessing
2. Analyze customer demographics (Gender, Age Group, Marital Status)
3. Identify top-performing states
4. Evaluate product category performance
5. Understand occupation-wise purchasing trends
6. Generate business insights for targeted marketing

##### Tools & Technologies Used:
1. Python
2. Jupyter Notebook
3. Pandas (Data Cleaning & Manipulation)
4. NumPy
5. Matplotlib
6. Seaborn

##### Project Workflow:
1. Data Loading
2. Data Cleaning (Handling missing values, duplicates, data types)
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Business Insight Generation

##### Key Insights Derived:

--> Female customers contributed higher overall sales compared to male customers.

--> Most purchases were made by the 26–35 age group.

--> Married women showed higher purchasing behavior.

--> Uttar Pradesh, Maharashtra, and Karnataka generated the highest revenue.

--> IT, Healthcare, and Aviation professionals were major spending groups.

--> Food, Clothing, and Electronics were top-selling categories.
